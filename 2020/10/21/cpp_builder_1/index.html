C++ Builder v10.2 如何生成可发布的可执行文件

a1phaboy
2020-10-21
C/C++, 磨刀, 编程

在网上找了很久，大多都是关于C++ Builder v6版本的说明，自己摸打滚爬了大半天，终于自己摸索出来了。。

Step 1

先创建一个工程

Step 2

右击工程，选择options

Step 3

选择所有选项（All configurations）

Step 4

选择C++ Compiler —> Classic Compiler 的参数设置为False

选择 C++ Linker —> Link with Dynamic RTL 参数设置为False

选择Runtime Packages —> Link with runtime packages 参数设置为False

点击编译生成（或按F9）后大功告成。

Tips

c++Builder的项目文件夹得绝对路径中不要包含中文或者是字符。
